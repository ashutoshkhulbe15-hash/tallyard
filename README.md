# Tallyard

Calculators for home improvement and DIY projects.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Hosted on Vercel

## Local development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Deploy

Push to `main` → Vercel auto-deploys.
